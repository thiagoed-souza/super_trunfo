# super_trunfo

🃏 Super Trunfo em C

Projeto desenvolvido em linguagem C como parte dos estudos em Análise e Desenvolvimento de Sistemas.

📌 Sobre o Projeto

Este projeto implementa uma versão simplificada do jogo Super Trunfo, permitindo:

Cadastro de cartas

Leitura de dados via terminal

Comparação de atributos

Exibição do vencedor

O objetivo é praticar:

Entrada e saída de dados (scanf e printf)

Estruturas condicionais

Estruturas de repetição

Organização de código em C

Boas práticas com Git e GitHub

🛠️ Tecnologias Utilizadas

Linguagem C

GCC (MinGW)

VS Code

Git e GitHub
